🔄 Sincronizar Vendas da Monetizze

Importe vendas diretamente da API da Monetizze para o painel. Útil para recuperar vendas que não foram capturadas automaticamente.


                            

                        

                        

                            💡 Como configurar:
                            

                                1. Acesse Monetizze → Ferramentas → API

                                2. Copie a X_CONSUMER_KEY (chave de API)

                                3. Acesse Railway → Seu Projeto → Variables

                                4. Adicione:

                                   • MONETIZZE_CONSUMER_KEY = sua chave X_CONSUMER_KEY

                                5. Aguarde o servidor reiniciar e volte aqui para sincronizar
